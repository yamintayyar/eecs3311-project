AI Chatbot

    The AI chatbot is context-aware chatbot integrated across the Login Dashboard, 
    Client Dashboard, Booking History and Payment pages. Its primary role is to guide 
    clients through the Service Booking Lifecycle, give information and provide navigational 
    help.

    - Can I cancel a booking that is already paid?
    - What are the different payment methods available?
    - What happens after I submit a booking request?

    Backend: Developed with Spring Boot using the Google GenAI SDK. It communicates with 
    the Gemini 2.5 Flash model via a REST Controller.

    Frontend: JavaScript implementation using the fetch API for asynchronous communication.

    Persistence: To ensure a seamless experience across the pages, the chat history is stored 
    in sessionStorage. This allows the conversation to continue uninterrupted as the client 
    moves between different dashboard tabs.

    All conversation data is stored only in the browser's volatile memory.
    The backend does not provide the AI with direct access to the database.
