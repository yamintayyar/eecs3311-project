# EECS 3311 Project 

## Setup Instructions

### Prerequisites
- Java 17+

---

### To run the Backend

```bash
cd backend/servicebooking
mvnw clean install
mvnw spring-boot:run
```

Test if the backend endpoints are working correctly by running the project then go to the url `http://localhost:8080/hello` in your browser.
